# Alquimista de Componentes

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad de Alquimista de Componentes para OpenClaw.

## Qué Resuelve
Transforma maquetas de interfaz de usuario en especificaciones de componentes reutilizables.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales, listos para producción
- Enfoque de seguridad primero
- Soporte de reversión

## Uso
/component-alchemist

## Ejemplo
```
Analizar el código en busca de problemas de diseño.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
